# student-management-vs-3

A beginner-friendly Student Management System built with Python, demonstrating file handling, search, delete, and student record management.

## Features
- Add Student
- Show Student List
- Search Student
- Delete Student
- Count Total Students

## Technologies Used
- Python
- File Handling (.txt)

## Purpose
This project was built to practice Python fundamentals, including loops, conditional statements, lists, and file handling through a simple student management system.
